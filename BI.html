Project 2: Broadband Speed Analytics: Residential Data - UK - March 2023

This analysis investigates UK broadband performance disparities using Ofcom's March 2023 dataset. Using Python for data cleaning, non-parametric statistical tests (Mann-Whitney U, Spearman’s ρ) and Power BI visualizations, it identifies rural-urban latency gaps, packet loss drivers and actionable infrastructure recommendations for ISPs
